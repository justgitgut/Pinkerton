Pinkerton Privacy Policy

Effective date: March 5, 2026

Pinkerton is a Chrome extension that helps users analyze activity on PinaLove pages. This policy explains what data Pinkerton processes, how it is used, and your choices.

1. Data We Process

Pinkerton processes data from pages you use on pinalove.com to provide core features.

  • Website content: profile fields, profile text, profile images/links, and related page content needed for scans and analysis.
  • User activity signals: scan timestamps, online/offline status over time, and derived activity trends.
  • Basic profile identifiers: username, profile URL, and profile metadata shown on the site.
  • Extension settings: scan interval, filter preferences, and optional comparison settings you enter.

2. How We Use Data

  • Show dashboard statistics and activity trends.
  • Enable filtering, navigation, and profile history features.
  • Support optional profile comparison visualizations.
  • Improve reliability (for example, migration and storage housekeeping).

3. Data Storage

Pinkerton stores data locally in your browser using chrome.storage.local. Data is retained only for extension functionality and can be removed by using the in-extension delete/clear option.

4. Data Sharing and Selling

Pinkerton does not sell personal data. Pinkerton does not transfer collected extension data to a separate developer backend for monetization.

5. Permissions

Pinkerton requests only permissions required for core operation:

  • alarms: run scheduled scans.
  • host permissions (pinalove.com): access relevant pages/endpoints.
  • scripting: inject packaged content scripts on matching pages.
  • storage: save local settings/history.
  • tabs: open/focus and coordinate extension workflows.

6. Security

Pinkerton follows Chrome Extension Manifest V3 constraints and keeps executable code bundled with the extension package.

7. Your Choices

  • You can stop scanning at any time in extension settings.
  • You can clear stored extension data at any time from the Settings tab.
  • You can uninstall the extension to remove future processing.

8. Changes to This Policy

This policy may be updated when Pinkerton features or compliance requirements change. The latest version will be published at this page URL.
